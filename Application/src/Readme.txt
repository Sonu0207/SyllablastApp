		SYLLABLAST
To initialize the game, follow these steps:
1. Copy the src directory to your desired location.
2. Open terminal and run the command 'npm run dev'
3. Go to http://localhost:3000/ in your web browser.


On the initial page, you will see options to select configurations. Choose one to start the game.

To perform a swap, click on two syllables whose positions you want to exchange.
Please note: Clicking on the same syllable twice will not increase the swap count.
